Graph Networks


Graph Neural Networks are a type of neural network that can be used to represent and understand data that is in the form of a graph. This is a very powerful tool, as many systems can be represented as graphs, such as social networks, molecular structures, and many more. We describe a graph by providing a set of nodes and edges, where the edges represent the connections between the nodes. It is possible for systems to have edges only in one direction, both directions and even self-connections (self-loops).

Figure 1: Four graphs. The two graphs on top are equivalent and the two on bottom are equivalent.


There is only a certain number of ways to arrange graphs, and we can easily quantify this. By using the adjacency matrix $A$, whose element $A_{ab}$ is the connection from node a to node b, we have a way to distinguish between graphs that is independent of the positions of each node. The adjacency matrix is of course diagonal for graphs that only have bi-directional edges
